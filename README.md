- `20260617_十五五规划推演报告.html`：十五五规划推演报告 · 建筑设计行业

## 上传后访问

如果仓库名为 `-html-reports`，GitHub Pages 开启后通常访问：

```text
https://你的GitHub用户名.github.io/-html-reports/
```

也可以直接访问单个报告：

```text
https://你的GitHub用户名.github.io/-html-reports/20260617_十五五规划推演报告.html
https://你的GitHub用户名.github.io/-html-reports/20260519_guangzhou_community_renewal_trend_report_v1.html
https://你的GitHub用户名.github.io/-html-reports/20260518_project_development_trend_analysis_report.html
https://你的GitHub用户名.github.io/-html-reports/20260518_community_renewal_aging_friendly_business_deployment_report.html
```

## 注意

以下报告使用了 Chart.js CDN：

- `20260617_十五五规划推演报告.html`
- `20260518_project_development_trend_analysis_report.html`

```text
https://cdn.jsdelivr.net/npm/chart.js@4.4.0/dist/chart.umd.min.js
```

浏览器能访问 jsDelivr 时，图表可正常显示。
