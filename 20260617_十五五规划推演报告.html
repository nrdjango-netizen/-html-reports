一、政策驱动的四大趋势

🏚️

趋势一：从"新城建设"转向"存量更新"

重点转向老旧小区、危旧房、城中村更新；老旧厂区/街区/交通枢纽改造；地下管网、生命线工程、平急两用设施等系统性更新。

对设计的直接含义
  • • 项目平均体量"碎片化+组合化"：几十万平米以下的片区改造
  • • 设计任务从"画新东西"变成"在复杂存量约束下做手术"
  • • 需掌握：结构安全、机电改造、消防、适老化、节能、风貌保护
🚫

趋势二：超高层受限，"好房子"成主流

强化"适用、经济、绿色、美观"，严格限制超高层建筑，提升住宅建筑设计品质。普通住宅、多层公建、社区级空间需求常态化。

对小所的利好
  • • 超高层和全国地标型项目本来就很难轮到小团队
  • • "好房子""完整社区""小而美公共空间"更易由中小事务所承接
  • • 项目数量大但单个合同金额中小，适合小团队产能
⚡

趋势三：绿色+数字化成为硬门槛

大力发展绿色建筑，推动超低能耗、低碳建筑规模化；推广装配式建筑和绿色施工；建立建筑能效等级制度；推进CIM平台建设。

对设计的含义
  • • "只会画方案"不够用了
  • • 需要把能耗、碳排、装配式构造、运维成本、CIM/BIM数据接口都考虑进设计
  • • 会用BIM/CIM、能做能耗模拟的团队报价更高、更易胜出
🔗

趋势四：设计服务走向"全过程咨询+运营导向"

建立"专项规划—片区策划—项目实施方案"实施体系，培育城市更新专业化运营机构。

高价值设计服务内涵
  • • 城市体检与诊断（问题发现与指标体系）
  • • 片区更新策划与城市设计
  • • 项目实施方案、投资测算与运营模式建议
  • • 后期绩效评估与运营空间优化

二、时间轴：2026 → 2030 风向演化

行业热度与机会分布预测
项目类型分布变化

阶段性风向详解

2026年
上游年 · 规则落地
政策状态
  • • 各地做城市体检
  • • 编制"十五五"城市更新专项规划
  • • 建立城市更新项目库
  • • 细化本地导则和操作规程
小所切入点
  • • 前期轻量服务：微更新策略、小区体检报告
  • • 小尺度样板段设计
  • • 与本地甲级院建立合作
风险：前期研究类项目合同金额小，回款节奏慢
2027年
中游年 · 实施高峰
政策状态
  • • 大规模启动老旧小区/城中村改造
  • • 管网更新、平急两用设施建设
  • • 招投标高峰到来
小所机会
  • • 单体与小尺度更新设计
  • • 大项目中的分项设计/专业分包
  • • 历史街区与工业遗产活化
风险：招投标门槛收紧，行业集中度上升
2028–2030年
下游年 · 行业洗牌
政策状态
  • • 法规体系、技术标准、收费机制基本成型
  • • 部分项目进入运营期和绩效评估期
  • • 需要运营优化再设计方案
企业分化方向
  • • 超大院：全国性城市更新总集成
  • • 区域综合院：深耕某省/都市圈
  • • 专业细分团队：某一领域全国口碑
小所两条路：A. 区域型小而美综合团队 / B. 全国可迁移专业细分专家

老旧小区/完整社区

老旧厂房/街区更新
厂房改文创/办公、老旧商业街更新、文旅IP场景植入
绿色与低碳改造专项
既有建筑节能改造、装配式内装方案设计
历史街区与文化遗产活化
历史建筑测绘、保护修缮、风貌控制与有机更新

🤝

3. 联合作战，别急着长大

合作形式
  • • 联合体投标：大院牵头，你们做方案创作与专题服务
  • • 框架协议：与平台公司签年度顾问合同
  • • 专家库/智库：进入城市更新专家库、三师联创团队
不要幻想直接抢总包，主动做"联合体成员+专业分包+智库角色"
中长期 3-5年
活得好 + 有主动权
📦

产品化

把项目经验提炼成可复制产品："老旧小区一站式微更新包""厂房改文创模板方案库"。带着方法论与成熟模板进入合作。

🗺️

片区视角

从单栋修补升级到片区统筹。在一条街、一个老厂区形成连续作品群，占据稳定话语权和议价空间。

💰

管好现金流

三看：看资金来源、看合同条款、看自身体量。跨期未结算收入控制在年营业额50%以内。

🏆

品牌专家化

核心合伙人在细分方向深耕，发表文章、参与行业会议。对外突出"专注于××城市更新"的定位。

风险矩阵与应对

风险类型 风险描述 影响程度 应对策略
资金风险 城市更新长周期、慢回款，政府和平台公司参与多 高 明确资金到位节点；合同按成果节点付款；控制跨期应收款比例
资质门槛 正式建设项目要求特定资质等级，小事务所达不到 中 联合体投标；做专业分包；进入专家库建立信任
行业集中 头部设计院通过一体化服务锁定大盘子项目 中 不抢总包，做细分方向的"必选项"
法律风险 规划变更、权属纠纷、补偿争议等 高 不做项目公司投资方；避免合同中承担过重"结果责任"
技术迭代 BIM/CIM/绿色建筑标准快速更新 中 定期法规与政策学习+技术培训；保持技术敏感度

核心洞察

🔄
结构性重塑而非增量
这轮城市更新不是"给所有人端上大锅饭"，而是一次集中考验：你能否在一个具体场景里做出别人离不开的专业与产品。
🎯
小所的核心优势
灵活创新+在地性理解+贴近生活场景。"好房子""完整社区""小而美公共空间"正是小团队的主场。
⏰
窗口期有限
2026-2027是卡位关键期。找准方向、搭好合作、做出案例，才能在2030年成为"必选项"。
💡
从"画图小馆"到"专家小所"
